Ajout de routes OSPF dans la table de routage

Grâce aux informations de plus court chemin déterminées par l’algorithme SPF, les chemins peuvent maintenant être ajoutés à la table de routage. La figure illustre les routes qui ont été ajoutées à présent à la table de routage IPv4 de R1.

La table de routage contient également toutes les routes et tous les réseaux connectés directement provenant d'autres sources, par exemple les routes statiques. Les paquets sont désormais acheminés en fonction de ces entrées dans la table de routage.

La figure présente la table de routage pour le routeur R-1. La table répertorie quatre réseaux directement connectés, sept réseaux distants, l'interface de sortie et le coût.
