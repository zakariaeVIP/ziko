Vérifier les paramètres d'interface OSPF

La méthode la plus rapide pour vérifier les paramètres d'interface OSPFv2 consiste à utiliser la commande show ip ospf interface. Cette commande permet d'obtenir une liste détaillée de toutes les interfaces activées conformément au protocole OSPFv2. Cette commande est utile pour déterminer si les instructions network ont été correctement composées.

Pour générer un résumé des interfaces compatibles OSPFv2, utilisez la commande show ip ospf interface brief, comme illustré dans la figure 1.

Utilisez le contrôleur de syntaxe de la Figure 2 pour générer et afficher un résumé des interfaces compatibles OSPFv2 sur R2 à l'aide de la commande show ip ospf interface brief. Notez que la spécification du nom de l'interface comme cela est fait dans la commande show ip ospf interface serial 0/0/1 fournit des informations OSPFv2 détaillées.

Continuez à utiliser le contrôleur de syntaxe de la Figure 2 pour générer un résumé des interfaces compatibles OSPFv2 sur R3 à l'aide de la commande show ip ospf interface brief. Récupérez et affichez des informations supplémentaires pour l'interface série 0/0/0 au moyen de la commande show ip ospf interface serial 0/0/0.

La figure présente le résultat de la commande suivante sur le routeur R1 : show ip OSPF interface brief. Le résultat répertorie les trois interfaces qui participent au processus OSPF, les adresses IP, les coûts et la zone OSPF.
