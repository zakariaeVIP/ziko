Mode de configuration OSPF du routeur

La Figure 1 représente la topologie de référence pour cette rubrique. OSPFv2 est activé via la commande router ospf process-id en mode de configuration globale. La valeur process-id représente un nombre compris entre 1 et 65 535. Elle est sélectionnée par l'administrateur réseau. La valeur process-id est une valeur locale. Autrement dit, elle ne doit pas être identique sur les autres routeurs OSPF pour établir la contigüité avec les voisins.

La Figure 2 montre un exemple d'activation du mode de configuration OSPFv2 sur le routeur R1.
