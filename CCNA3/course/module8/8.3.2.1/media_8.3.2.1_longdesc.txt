La Figure 1 illustre la topologie de référence pour le protocole OSPF avec l'adressage IPv6 au lieu de l'adressage IPv4 utilisé précédemment avec OSPFv2. La Figure 2 affiche les commandes nécessaires pour configurer l'adressage et OSPFv3 sur le routeur R1.   La figure commence en mode de configuration globale et la première commande est ipv6. unicast-routing.  L'étape suivante consiste à passer à une interface spécifique avec la commande interface gigabit ethernet 0/0.  La commande suivante est description R1 LAN.  La commande suivante est ipv6 address 2001:DB8:CAFE:A001::1/64. Cette commande configure l'adresse IPv6 sur l'interface.  L'interface est alors activée avec la commande « no shut ».  La figure continue avec la première interface série. La première commande est interface serial 0/0/0.  La commande suivante est description link to R2.  La commande suivante est ipv6 address 2001:DB8:CAFE:A001::1/64. Cette commande configure l'adresse IPv6 sur l'interface.  L'interface est alors activée avec la commande « no shut ».  La figure poursuit avec la deuxième interface série. La première commande est interface serial 0/0/1.  La commande suivante est description link to R3.  La commande suivante est ipv6 address 2001:DB8:CAFE:A001::1/64. Cette commande configure l'adresse IPv6 sur l'interface.  L'interface est alors activée avec la commande « no shut ». La Figure 3 répertorie les six étapes de configuration d'OSPFv3.  La première étape consiste à activer le routage unicast IPv6 avec la commande ipv6  unicast-routing.  La deuxième étape est facultative. Lors de cette étape, les adresses link-local sont configurées.  La troisième étape consiste à configurer un ID de routeur 32 bits dans le mode de configuration de routeur OSPFv3 à l'aide de la commande router-id.  La quatrième étape consiste à configurer les informations facultatives de routage, telles que l'ajustement de la bande passante de référence.  La cinquième étape est facultative. Lors de cette étape, les paramètres spécifiques à l'interface OSPFv3 sont configurés, tels que la bande passante.  La sixième étape consiste à activer le routage IPv6  via la commande ipv6 OSPF area.
