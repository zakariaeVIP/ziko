La Figure 1 représente un tableau répertoriant les structures de données OSPF.  OSPF crée une base de données de contiguïté, qui contient la table des voisins et affiche tous les routeurs voisins.  La commande show ip OSPF neighbor, permet d'afficher la table.  OSPF crée une base de données d'état de liens, qui contient la table topologique, ainsi que des informations sur tous les autres routeurs du réseau.  La commande show ip OSPF database, permet d'afficher cette table.  OSPF crée également une base de données de réacheminement qui contient la table de routage.  Elle répertorie les routes générées lors de l'exécution d'un algorithme sur la base de données d'états de liens.  La commande, show ip route, permet d'afficher cette table.
