Vérifier les interfaces OSPFv3

La méthode la plus rapide pour vérifier les paramètres d'interface OSPFv3 consiste à utiliser la commande show ipv6 ospf interface. Cette commande permet d'obtenir une liste détaillée de toutes les interfaces activées conformément au protocole OSPFv3.

Pour extraire et afficher un résumé des interfaces compatibles OSPFv3 sur R1, utilisez la commande show ipv6 ospf interface brief, comme illustré dans la Figure 1.

Utilisez le contrôleur de syntaxe de la Figure 2 pour afficher un résumé des interfaces compatibles OSPFv3 sur R2 à l'aide de la commande show ipv6 ospf interface brief. Notez que la spécification du nom de l'interface comme cela est fait dans la commande show ipv6 ospf interface serial 0/0/1 fournit des informations OSPFv3 détaillées.

Continuez à utiliser le contrôleur de syntaxe de la Figure 2 pour générer un résumé des interfaces compatibles OSPFv3 sur R3 à l'aide de la commande show ipv6 ospf interface brief. Récupérez et affichez des informations supplémentaires pour l'interface série 0/0/0 au moyen de la commande show ipv6 ospf interface serial 0/0/0.

La Figure 1 présente le résultat de la commande suivante sur le routeur R1 : show ipv6 OSPF interface brief. Le résultat est une liste détaillée pour chaque interface OSPF. La Figure 2 illustre un élément multimédia interactif qui permet à l'étudiant de vérifier les interfaces OSPFv3 sur le routeur R2.
