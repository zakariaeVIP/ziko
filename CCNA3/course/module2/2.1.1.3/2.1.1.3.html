Annonces VTP

VTP comprend trois types d'annonces :

  • Annonces de type résumé - Elles informent les commutateurs contigus du nom de domaine et du numéro de révision de la configuration du VTP.

  • Annonces de type requête - Elles sont envoyées en réponse à un message d'annonce de type résumé lorsque celui-ci contient un numéro de révision de configuration supérieur à la valeur actuelle.

  • Annonces de type sous-ensemble - Elles contiennent les informations sur le VLAN, y compris les éventuelles modifications.

Par défaut, les commutateurs Cisco émettent des annonces de type résumé toutes les cinq minutes. Les annonces de type résumé communiquent le nom du domaine VTP actuel et le numéro de révision de la configuration aux commutateurs adjacents.

Le numéro de révision de la configuration est un nombre de 32 bits qui indique le niveau de révision d’un paquet VTP. Chaque commutateur VTP effectue un suivi du numéro de révision de configuration VTP qui lui est attribué.

Ces informations permettent de déterminer si les données reçues sont plus récentes que la version actuelle. Chaque fois que vous modifiez les informations VLAN d'un commutateur VTP, le numéro de révision de la configuration augmente de 1.

Remarque : pour réinitialiser une révision de la configuration sur un commutateur, changez le nom du domaine VTP, puis réattribuez-lui son nom d'origine.

Lorsque le commutateur reçoit un paquet d’annonces de type résumé, il compare le nom du domaine VTP à son propre nom de domaine VTP. Si les noms sont différents, le commutateur ignore simplement le paquet. S'ils sont identiques, le commutateur compare le numéro de révision de la configuration à son propre numéro. Si son numéro est supérieur ou égal au numéro de révision de la configuration du paquet, il ignore le paquet. Si son propre numéro de révision de la configuration est inférieur, il envoie une annonce de type requête pour demander une annonce de type sous-ensemble.

L'annonce de type sous-ensemble contient les informations VLAN ainsi que toutes les modifications. Lorsque vous ajoutez, supprimez ou modifiez un réseau local virtuel sur le serveur VTP, ce dernier incrémente le numéro de révision de la configuration et émet une annonce de type résumé.
