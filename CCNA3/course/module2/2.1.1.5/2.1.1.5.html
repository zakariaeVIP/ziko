Configuration VTP par défaut

La commande EXEC privilégiée show vtp status affiche l'état du VTP. L'exécution de cette commande sur un commutateur de la gamme Cisco 2960 Plus génère la sortie indiquée dans la figure.

Vous trouverez ci-après une brève description du résultat de la commande pour les paramètres de show vtp status.

VTP Version capable and running

  • Le premier paramètre affiche la version VTP que le commutateur est en mesure d'exécuter et le second celle qu'il exécute actuellement.

  • Par défaut, les commutateurs implémentent la version 1.

  • Si la plupart d'entre eux prennent en charge la version 2, les commutateurs plus récents prennent également en charge la version 3.

VTP Domain Name

  • Nom identifiant le domaine administratif du commutateur.

  • Par défaut, le nom du domaine VTP est laissé vide (valeur NULL).

VTP Pruning Mode

  • Indique si l'élagage est activé ou désactivé.

  • Par défaut, l'élagage VTP est désactivé.

VTP Traps Generation

  • Indique si des traps VTP sont envoyés à un poste de gestion du réseau.

  • Par défaut, les traps VTP sont désactivés.

Device ID

  • Adresse MAC du commutateur.

Configuration Last Modified

  • Date et heure de la dernière modification de la configuration.

  • Affiche l’adresse IP du commutateur ayant entraîné la modification de configuration dans la base de données.

VTP Operating Mode

  • Il peut s'agir du mode serveur, client ou transparent.

  • Par défaut, un commutateur fonctionne en mode serveur VTP.

Maximum VLANs Supported Locally

  • Le nombre de VLAN pris en charge varie selon les plates-formes de commutateurs.

Number of Existing VLANs

  • Intègre le nombre de VLAN par défaut et configurés.

  • Le nombre par défaut de VLAN existants varie selon les plates-formes de commutateurs.

Configuration Revision

  • Numéro actuel de révision de la configuration sur ce commutateur.

  • Le numéro de révision de est un nombre de 32 bits qui indique le niveau de révision d’une trame VTP.

  • Le numéro de configuration par défaut d’un commutateur est zéro.

  • Chaque fois qu’un réseau local virtuel est ajouté ou supprimé, le numéro de révision de configuration est incrémenté.

  • Chaque commutateur VTP effectue un suivi du numéro de révision de configuration VTP qui lui est attribué.

MD5 Digest

  • Somme de contrôle de 16 octets de la configuration VTP.
