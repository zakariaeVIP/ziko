Les problèmes de configuration du commutateur de la couche 3

Les problèmes communs au routage inter-VLAN existant et au routage inter-VLAN router-on-a-stick surviennent également dans le cadre de la commutation de couche 3. Pour résoudre des problèmes de commutation de couche 3, il convient de vérifier les éléments suivants :

  • VLAN : les VLAN doivent être définis sur tous les commutateurs. Les VLAN doivent être activés sur les ports trunk. Les ports doivent se trouver dans les VLAN appropriés.

  • SVI : Les SVI doivent disposer de l'adresse IP ou du masque de sous-réseau correct. Les interfaces SVI doivent être opérationnelles. Chaque interface SVI doit correspondre au numéro du VLAN.

  • Routage : le routage doit être activé. Chaque interface ou réseau doit être ajouté au protocole de routage ou des routes statiques doivent être saisies, selon le cas.

  • Hôtes : les hôtes doivent avoir l'adresse IP ou le masque de sous-réseau correct. Les hôtes doivent avoir une passerelle par défaut associée à une interface SVI ou un port routé.

Pour résoudre les problèmes de commutation de couche 3, familiarisez-vous avec la mise en œuvre et la conception de la topologie.
