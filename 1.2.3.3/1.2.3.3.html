Commandes de base de l'ILC du routeur

Une configuration de routeur de base inclut le nom d'hôte pour identification, des mots de passe pour la sécurité, l'affectation d'adresses IP à des interfaces pour la connectivité et un routage de base. La figure 1 illustre les commandes à saisir pour activer RIPv2 sur un routeur. Vérifiez et enregistrez les modifications de configuration, au moyen de la commande copy running-config startup-config. La figure 2 illustre les résultats des commandes de configuration saisies dans la figure 1. Pour effacer la configuration du routeur, utilisez la commande erase startup-config, puis la commande reload.

Utilisez le contrôleur de syntaxe de la figure 3 pour vous exercer à la configuration de base d'un routeur.

La figure 1 présente un exemple de configuration de routeur pour le protocole RIPv2. La figure 2 présente le résultat de la commande « show running dash config » sur un routeur. La Figure 3 est une activité interactive qui permet aux stagiaires de s’entraîner à configurer les paramètres de base d'un routeur.
